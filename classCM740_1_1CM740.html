Detailed Description

The main class that others will use to interact with the CM740 and attached devices.

This is the main access point for all users of this CM740 driver. Note that it is build for a little endian system, and if it is used on a big endian system, the code will need to be reviewed. This is because it is reading the 2 byte values as they are on the CM740 (which is little endian).
